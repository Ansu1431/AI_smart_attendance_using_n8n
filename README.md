# 📖 Overview

The **AI Smart Attendance System** is an intelligent workflow automation project developed using **n8n**. It automates attendance management by integrating AI services, APIs, and cloud-based workflows, reducing manual effort while improving accuracy and efficiency.

The project demonstrates how modern no-code/low-code automation can be used to build smart attendance solutions for schools, colleges, offices, and organizations.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Ansu1431/AI_smart_attendance_using_n8n.git
```

Move into the project

```bash
cd AI_smart_attendance_using_n8n
```

---

# 📥 Import Workflow

1. Open **n8n**
2. Click **Import Workflow**
3. Select the JSON workflow
4. Configure credentials
5. Activate Workflow
