# Setup G Sheets Sync\n\n## 1. Google Service Account\n1. [Google Cloud Console](https://console.cloud.google.com/apis/library/sheets.googleapis.com)\n2. Enable Sheets API → Credentials → Create Service Account → JSON key\n3. **PASTE JSON** into `data/gsheet_credentials.json`\n4. **Share G Sheet** with service account `client_email`\n\n## 2. Update config/email_config.py\n```\nGOOGLE_SHEET_ID = 'docs.google.com/spreadsheets/d/THIS_PART'\nGOOGLE_SHEET_RANGE = 'Sheet1!A8:H' # name,email,parent,subject,exam_type,term,score,max\n```\n\n## 3. Columns in Sheet\n| A:name | B:student_email | C:parent_email | D:subject | E:exam_type | F:term | G:score | H:max_score |\n| komal  | ...             | ...            | maths     | semester    | 1      | 85      | 100         |\n\n## 4. Test\npip install -r requirements.txt\npython app.py\nlocalhost:5000/admin → Sync G Sheet → /profile/komal
